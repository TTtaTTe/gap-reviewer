# 채점 기준표 (Scoring Rubric)

## Phase 1: 지시서 분석 신뢰도

| 항목 | 배점 | 기준 |
|------|------|------|
| 완전성 | 2.5 | 지시서의 모든 주요 요구항목이 빠짐없이 추출되었는가 |
| 정확성 | 2.5 | 추출된 항목이 원문의 의도와 일치하는가 |
| 구체성 | 2.5 | 모호하지 않고 Phase 2에서 검증 가능한 수준인가 |
| 분류 체계 | 2.5 | 항목이 논리적으로 분류/구조화되었는가 |
| **합계** | **10.0** | |

## Phase 2: 작업물 충족도

### 점수 기준

| 점수 | 상태 | 의미 |
|------|------|------|
| 10.0 | 충족 | 요구사항을 정확히 반영. 보완 불필요 |
| 7.0~9.9 | 대부분충족 | 경미한 보완 필요. 핵심 내용은 포함됨 |
| 4.0~6.9 | 부분충족 | 핵심은 있으나 상당 부분 미흡. 보완 필요 |
| 1.0~3.9 | 미흡 | 언급은 있으나 요구사항과 거리가 있음. 대폭 보완 필요 |
| 0.0 | 누락 | 해당 항목 자체가 없음. 신규 작성 필요 |

### 2회 반복 결과 처리

| 규칙 | 설명 |
|------|------|
| 항목별 점수 | 2회 중 **낮은 점수** 채택 (보수적 평가) |
| 종합 점수 | 전체 항목 점수의 가중 평균 (필수 항목 1.5배 가중) |
| 공통 지적 | 2회 모두 지적 → **[공통]** 태그 (신뢰도 높음) |
| 단독 지적 | 1회만 지적 → **[참고]** 태그 (확인 권장) |

## 평가 원칙 (모든 에이전트 공통)

### DO (해야 할 것)
- 주요 요구항목의 누락 여부를 확인한다
- 표현이 달라도 내용이 동일하면 충족으로 판단한다
- 구체적인 보완 방향과 작성 예시를 함께 제시한다
- 필수 항목과 권장 항목을 구분하여 평가한다

### DON'T (하지 말아야 할 것)
- 단어의 어감이나 표현 방식을 지적하지 않는다
- 문체 통일 여부를 평가하지 않는다
- 사소한 표현 차이로 감점하지 않는다
- 지시서에 없는 항목을 임의로 추가하여 평가하지 않는다
- 완벽한 문장을 요구하지 않는다
